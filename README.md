## Setup Commands

1. clone this project
````
$ git clone https://github.com/jangidp318/solvevolve-react-native-challenge.git
````
2. Install the Expo Cli

3. Open the project directory 

4. Install the expo project
````
$ npm install
````
5. Run the project
````
$ expo start
````
